Crucible Beta — macOS
======================

You need: Python 3 and Node.js 18+ (for Claude MCP). Install from python.org / nodejs.org or Homebrew.

1. Unzip this folder anywhere (keep all files together).

2. First time only — do NOT double-click normally:
   Control-click "Install-Crucible.command" → Open → click "Open" in the dialog.
   (Plain double-click may say the developer cannot be verified.)

   If Terminal says the file is quarantined:
   xattr -d com.apple.quarantine /path/to/Install-Crucible.command

3. Read the output in Terminal: your Beta token and ChatGPT connector link are at the end.
   Press Enter to close.

4. Fully quit and reopen Claude Desktop so MCP loads.
