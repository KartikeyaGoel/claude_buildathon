Crucible Beta — Windows
========================

You need: Python 3.9+ and Node.js 18+ (for Claude MCP). https://python.org and https://nodejs.org

1. Unzip this folder anywhere (keep all files together).

2. Double-click: RunInstaller-hidden-log.vbs
   - If Windows SmartScreen warns: "More info" → "Run anyway"
   - If blocked: right-click the file → Properties → check "Unblock" → OK, try again

3. Notepad opens with install output. Copy your Beta token and follow the ChatGPT steps if needed.

4. Fully quit and reopen Claude Desktop so MCP loads.

Problems: see the log file path in Notepad title or %TEMP%\crucible-mcp-install.log
